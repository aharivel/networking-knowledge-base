# Networking Knowledge Base

> *From Core Networking to Cloud Routing & Tunneling*

## 📚 About

This is a living knowledge base for networking concepts, protocols, and technologies. It serves as a personal reference and learning resource that I update as I explore new topics.

## 🗂️ Structure

The knowledge base is organized into **5 main sections**:


| Section | Focus Area | Key Topics |
|---------|-----------|------------|
| **01: Core Networking** | Fundamentals | OSI Model, TCP/IP, Subnetting, DNS, DHCP |
| **02: High Performance** | Low-latency | DPDK, RDMA, eBPF, XDP |
| **03: Container Networking** | Containers & Orchestration | Docker, CNI, Calico, Cilium, VXLAN |
| **04: VM Networking** | Virtualization | Open vSwitch, SR-IOV, Virtual Switches |
| **05: Cloud & Tunneling** | Cloud-native | VPC Peering, VPN, Service Mesh, WireGuard |

## 🎯 How to Use

- **Browse by section** using the sidebar
- **Search** for specific topics
- **Navigate** between related articles
- **View diagrams** - all architecture and flow diagrams use [Mermaid.js](https://mermaid.js.org/)

## 🛠️ Technical Stack

- **Static Site**: Docsify (no build step, just markdown files)
- **Diagrams**: Mermaid.js for flow/architecture visualizations
- **Hosting**: Cloudflare Pages (static site hosting with Git integration)
- **Version Control**: Git
